# Evaluación Final - Módulo 4: Consumo de APIs y POO

## 🎯 Objetivos de la Aplicación
El objetivo principal de este proyecto es poner en práctica el consumo de servicios web a través de peticiones HTTP, la manipulación de datos en el cliente y la implementación de la Programación Orientada a Objetos (POO) en JavaScript. 

Específicamente, la aplicación resuelve:
1. **Consumo y Optimización de la API de Rick and Morty:** * Obtener y listar información mediante `fetch()`.
   * Implementar un sistema de almacenamiento local (caché en memoria) para optimizar el consumo de datos y evitar llamadas innecesarias al servidor a partir de la segunda consulta.
   * Agrupar y transformar los datos de los personajes utilizando métodos avanzados de arrays (`.map()` y `.reduce()`).

2. **Administrador de Usuarios con AJAX:**
   * Desarrollar una clase en JavaScript que administre datos obtenidos desde el servicio web `JSONPlaceholder` utilizando `XMLHttpRequest`.
   * Crear métodos específicos para listar usuarios, buscar información básica, avanzada o de dirección mediante `prompt()`, y ordenar alfabéticamente los registros.
